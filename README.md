###在oc中调用swift文件方法
   1. 在工程中导入swift的类
   2. 创建全局预编译文件 PrefixHeader.pch
   3. 在预编译文件PrefixHeader.pch中 添加 #import ‘项目名-Swift.h")
   4. 然后就可以在oc中使用swift的类了
